## About App

Simple famous quote quiz app backend

## Start App

Clone repo to directory
set correct database details at .env file
```bash
composer install
```
```bash
php artisan migrate
```
```bash
php artisan db:seed
```
```bash
php artisan serve
```
admin panel at: /admin
admin account: test@test.com password

[documentation](https://laravel.com/docs)
